# 🇮🇳 India District Crime Analysis (2014–2023)

A full exploratory and predictive analysis of district-level crime data across India.

## 📊 Features

- 30,000+ records
- 12 states
- 10 crime categories
- 10 years of data
- Time-aware validation
- Forecasting model
- Anomaly detection

## 🔍 Techniques Used

- Feature Engineering
- XGBoost Regression
- Time-Series Lag Modeling
- Residual Diagnostics
- Clustering
- Z-Score Anomaly Detection

## 📈 Results

- Time-based validation
- Autoregressive modeling
- Near-perfect lag-based predictions
- Structured anomaly detection

## ⚠️ Note

This dataset is synthetic and structured for ML experimentation.

Made with Python, Pandas, Scikit-learn, XGBoost.
