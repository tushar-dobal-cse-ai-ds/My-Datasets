# Student Performance Dataset

## Description
This dataset contains information related to student demographics, academic background,
and performance. It can be used for data analysis, visualization, and machine learning
tasks such as regression and classification.

## Columns
Refer to the data_dictionary.xlsx file for detailed column descriptions.

- Exploratory Data Analysis (EDA)
- Student performance prediction
- Machine learning practice
- Educational analytics

identifiable information.
